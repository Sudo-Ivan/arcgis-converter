ArcGIS Layer Converter

Convert ArcGIS Feature Layers to various formats for offline use.

Input Layer

Enter either a Feature Server URL or a specific layer URL ending with the layer index (e.g., /0, /1).

Select Layers

Export Options

Load a layer to see export options.

Layer Preview

Load a layer to see the preview.

Supports formats compatible with ATAK, QGIS, Google Earth, and more.

URL Parameters

You can use URL parameters to pre-load layers and optionally export them:

?url=LAYER_URL - Load a single layer
?url=FEATURE_SERVER_URL - Load a Feature Server and show layer selection
?urls=LAYER_URL1,LAYER_URL2 - Load multiple specific layers
?export=FORMAT - Export in specified format (kml, geojson, gpx)

Note: The URLs must be properly encoded. For example:

https://arcgis-converter.vercel.app/?url=https%3A%2F%2Fexample.com%2Farcgis%2Frest%2Fservices%2FExampleLayer%2FFeatureServer
https://arcgis-converter.vercel.app/?urls=https%3A%2F%2Fexample.com%2Farcgis%2Frest%2Fservices%2FLayer1%2FFeatureServer%2F0%2Chttps%3A%2F%2Fexample.com%2Farcgis%2Frest%2Fservices%2FLayer2%2FFeatureServer%2F1
